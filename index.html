Conceitos básicos do HTML

O que é o HTML?

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é uma linguagem de marcação usada para criar e estruturar páginas web. Ele consiste em uma série de elementos ou "tags" que envolvem o conteúdo da página para indicar ao navegador como exibi-lo. As tags são compostas por códigos entre colchetes, como <tag>. Cada tag tem uma função específica, como definir títulos, parágrafos, listas, links e outros elementos da página.

Principais Tags

As tags <H1>, <H2>, <H3>, <H4>, <H5>, e <H6> são tags HTML usadas para definir títulos ou cabeçalhos em uma página web. Cada uma delas representa um nível de importância, sendo <H1> o mais importante e <H6> o menos importante.

Exemplo de uso:

<H1> Título Principal </H1>

<H1> Subtítulo <H2>
