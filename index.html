Conceitos básicos do HTML

O que é o HTML?

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é uma linguagem de marcação usada para criar e estruturar páginas web. Ele consiste em uma série de elementos ou "tags" que envolvem o conteúdo da página para indicar ao navegador como exibi-lo. As tags são compostas por códigos entre colchetes, como <tag>. Cada tag tem uma função específica, como definir títulos, parágrafos, listas, links e outros elementos da página.

Principais Tags

As tags <H1>, <H2>, <H3>, <H4>, <H5>, e <H6> são tags HTML usadas para definir títulos ou cabeçalhos em uma página web. Cada uma delas representa um nível de importância, sendo <H1> o mais importante e <H6> o menos importante.

Exemplo de uso:

<H1> Título Principal </H1>

<H2> Subtítulo <H2>

A tag <p> em HTML é usada para definir parágrafos de texto em uma página web. A palavra "p" representa "parágrafo". Essa tag é usada para agrupar um bloco de texto e indicar ao navegador que esse bloco deve ser tratado como um parágrafo separado.

Exemplo de uso:

<p>Isto é um parágrafo. O texto dentro desta tag será exibido como um bloco separado.</p>

<p>Outro parágrafo aqui. Cada tag <p> cria um novo parágrafo na página.</p>

A tag <mark> em HTML é usada para destacar parte de um texto, geralmente aplicando um marcador de cor de fundo amarelo ao texto contido dentro da tag. Essa marcação é útil para chamar a atenção do leitor para informações específicas.

Exemplo de uso:

<p>Este é um <mark>texto marcado</mark> em uma frase.</p>

<p>Este é um texto marcado em uma frase.</p>

A tag <small> em HTML é usada para indicar que o texto contido dentro dela deve ser exibido em um tamanho de fonte menor do que o texto circundante. Essa tag é usada para indicar que o texto é de menor importância, como notas de rodapé, legendas ou informações secundárias.

Exemplo de uso:

<p>Isso é um parágrafo regular.</p>

<small>Esta é uma nota de rodapé ou texto de menor importância.<small>

A tag <i> em HTML é usada para aplicar ênfase ou itálico ao texto. Anteriormente, ela era comumente utilizada para indicar que o texto deveria ser exibido em itálico, mas, de acordo com as práticas atuais de desenvolvimento web, é mais apropriado usar a tag <em> para enfatizar o texto e a propriedade CSS para controlar o estilo, incluindo a inclinação.

Exemplo de uso:

<p>Este é um texto normal.</p>

<p><i>Este texto está em itálico.<i></p>

A tag <u> em HTML é usada para sublinhar o texto. Quando você envolve um trecho de texto com a tag <u>, o navegador exibirá esse texto com uma linha horizontal sob ele, indicando que está sublinhado.

Exemplo de uso:

<p>Este é um texto normal.</p>

<p><u>Este texto está sublinhado.</u></p>
